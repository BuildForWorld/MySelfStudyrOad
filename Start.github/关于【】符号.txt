[] 是偏移，但a[1]并不是a的地址+1，a和a[0]也不一样，a是一个首地址，a[0]是a数组第一个元素内容，a【1】是第二个元素的内容。
与二级指针关系：
	int demo (char **B)
	{
	printf("%s \n",B[0]);
	return 0;

	}
	void main()
	{
	char *a[3] = {"aaaa","bbbb","cccc"};
	demo (a);
	} 
可以输出“aaaa”，但把B【0】该为b，就不能输出了，所以B【0】等价于 * B，B【1】等价于*（B+1）。B+1 的意思是下一个元素的地址，加了一个元素所占内存，并不是单纯加了一。
